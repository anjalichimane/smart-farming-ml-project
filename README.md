# 🌱 Smart Farming System

An AI-powered web application that helps farmers make smart decisions by providing:

- 🌾 Crop Recommendation  
- 🧪 Fertilizer Suggestion  
- 🦠 Plant Disease Detection  

---

## 🚀 Features

✔ Predict best crop based on soil & weather data  
✔ Suggest suitable fertilizers  
✔ Detect plant diseases using image input  
✔ User authentication (Login / Signup)  
✔ Clean and user-friendly UI  

---

## 🛠 Tech Stack

### 💻 Frontend
- React.js
- HTML, CSS, JavaScript

### ⚙ Backend
- FastAPI (Python)

### 🤖 Machine Learning
- Scikit-learn (Crop & Fertilizer models)
- TensorFlow (Disease Detection - EfficientNet)

### 🗄 Database
- MongoDB (User Authentication)
